Название: Начальная схема БД и миграции

Требования

- Добавить миграции (Flyway/Liquibase) для таблиц: users, zones, checkpoints, keys.
- Определить базовые индексы (по id, code, created_at по необходимости).
- Обновить конфигурацию подключения.

Тесты

- Интеграционный тест миграций (поднимаем контейнер PostgreSQL и применяем миграции без ошибок).

Документация

- DDL-черновик в docs/schema.md и диаграмма (простая текстовая нотация).

Готово, когда

- Миграции применяются без ошибок.
- Тесты зелёные.
- Документация обновлена.
