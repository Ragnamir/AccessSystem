Название: Реестр ключей пунктов и проверка подписи события

Требования

- Таблица ключей используется для получения публичного ключа по checkpointId.
- Валидация подписи события (ECDSA/RSA) по канонической форме.
- Обработка ошибок подписи.

Тесты

- Позитив/негатив: корректная подпись, неверный ключ, повреждённый payload.

Документация

- Описание формата подписи и канонизации в docs/crypto.md.

Готово, когда

- Подпись проверяется корректно; ошибки обрабатываются; тесты зелёные.
