Название: Базовый endpoint /ingest/event (валидация + 202)

Требования

- Реализовать `POST /ingest/event` с JSON-схемой из Overview.
- Валидация входа (структура, обязательные поля, формат timestamp).
- Ответ `202 Accepted` и базовый лог событий.

Тесты

- Контрактные/интеграционные тесты endpoint: валид/невалид payload.

Документация

- OpenAPI спеку обновить и положить в docs/openapi.yaml.

Готово, когда

- Endpoint отвечает 202 для валидных запросов, 4xx для ошибок.
- Тесты зелёные, спека актуальна.
