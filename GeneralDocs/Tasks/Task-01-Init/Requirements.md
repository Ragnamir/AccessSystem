Название: Инициализация проекта и окружения

Требования

- Создать проект на Java (Spring Boot) с модулями: api, core (по необходимости — моно-модуль допустим на старте).
- Поднять PostgreSQL через docker-compose, настроить подключение.
- Добавить профили окружений (dev, test).
- Добавить health endpoint.
- Настроить CI-скрипт (скелет) для сборки и запуска тестов.

Тесты

- Интеграционный тест health endpoint (200 OK).
- Тест подключения к БД (минимальная проверка через миграции).

Документация

- README с инструкцией запуска (Docker, профили).
- Описание структур модулей и зависимостей.

Готово, когда

- Сервис стартует локально, health: OK.
- Тесты зелёные.
- Документация актуальна.
