Название: Генератор событий (отдельный модуль)

Требования

- Отдельное приложение/модуль, генерирующее события с настраиваемой частотой.
- Приложение стартует первым, подбирает креды из файла для доступа к бд. генерирует данные в БД (пользователи, пропускные пункты, зоны, ключи), если необходимо.
- Сценарии: корректные переходы, неправильные подписи, повтор событий, запрет доступа.
- Отправка на /ingest/event.

Тесты

- Юнит-тесты генерации сценариев; smoke-тест интеграции с тестовым сервером.

Документация

- README по запуску генератора, параметры CLI/конфига.

Готово, когда

- Генератор отправляет события согласно сценариям; тесты зелёные.
